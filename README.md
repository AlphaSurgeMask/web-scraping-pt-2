# Web scraping part 2
## Install required npm packages
```
npm install
```
or
```
npm install cheerio

npm install axios

npm install json2csv

npm install fs

npm install puppeteer
```

## Also run the following for headless chromium

```
sudo apt update

sudo apt upgrade

sudo apt install -y \
  libnspr4 \
  libnss3 \ 
  libatk1.0-0 \
  libatk-bridge2.0-0t64 \
  libcups2 \
  libxkbcommon-x11-0 \
```
